## 🧠 Study 진행 방식

- **준비**: 각자 주제별로 공부한 내용을 Markdown 문서로 정리 해오기
- **발표**: 인당 5~10분 | 발표 자료는 정리해온 문서
  - 발표 자료에 대해서는 Pull Requeset 생성 및 멤버 `Reviewer`로 추가
- **Q&A**: 발표 후 질문 & 토론 (모르는 부분은 함께 찾아보기)
  - 논의 내용 PR에 comment 남기기
